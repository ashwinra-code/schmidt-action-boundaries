# Data Use

This repository contains package-local source artifacts and locked canonical objects needed to reproduce the retrospective AFC proof-of-concept analysis.

## Included Inputs

- `data/raw/source_artifacts/cross_model/inference/`: per-problem cross-model JSON outputs used to rebuild the trial-level response table.
- `data/raw/source_artifacts/cross_model/DataOriginal_FINAL.csv`: cross-model scenario metadata.
- `data/raw/source_artifacts/gpt_health/DataOriginal_FINAL.csv`: GPT-Health structured source table.
- `data/canonical/`: locked canonical workbench objects used as comparators or secondary frozen artifacts.

## Use Boundary

These files are included to support forensic replication of the retrospective proof-of-concept. They are not a clinical dataset, medical device, or prospective validation corpus.

The strict runner recomputes the primary AFC result and shuffle null from package-local source artifacts. Some secondary robustness summaries remain locked canonical objects unless explicitly regenerated by the corresponding script.
